# Launch Checklist

- [ ] Apply all Supabase migrations.
- [ ] Set production environment variables.
- [ ] Run `npm run check:prod-env`.
- [ ] Run `npm run format:check`.
- [ ] Run `npm run typecheck`.
- [ ] Run `npm run lint`.
- [ ] Run `npm test`.
- [ ] Run `npm run build`.
- [ ] Configure Supabase auth URLs.
- [ ] Configure Paymob integrations and webhook URL.
- [ ] Configure custom domain DNS and SSL.
- [ ] Verify `/p/launch-os`, `/sitemap.xml`, and `/robots.txt`.
- [ ] Test registration, login, AI generation, publish, analytics consent, and billing checkout.
- [ ] Confirm error logs are connected to production log drains.
